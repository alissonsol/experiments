## Porkbun

- Need to enable API access for accounts (can be done in bulk).
- Need to get API access key and place in file formatted as [porkbun.template.json](./porkbun.template.json)
  - Visit <https://porkbun.com/account/api>
  - Copy `porkbun.template.json` to `porkbun.json` and fill in your keys
- Reference: [porkbun-cli](https://github.com/ragelink/porkbun-cli) code.
